About this project

This Three.js project is an interactive 3D architectural visualization recreating Anderson Mancini's original masterpiece.

The 3D model is built with high-fidelity meshes, realistic water reflection shaders, dynamic shadow mapping, and animated bird assets flying in real time.

Optimized for modern WebGL capabilities using orbit coordinate navigation and canvas screencast algorithms.
